# BPNeuronNetWork
**C++语言结合openCV、MNIST实现的BP神经网络手写数字识别系统（大概）**
~~其实就是弟弟东拼西凑来的东西~~
##网络结构
>输入层784个神经元,1个隐含层100个神经元,输出层10个神经元

##图像预处理
>只会最简单的归一化、二值化
